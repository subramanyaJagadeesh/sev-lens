# OpsPulse AI

OpsPulse AI is a stage-driven incident response demo.

## Current Focus

- **V1:** complete and preserved as the baseline reference.
- **V2:** active development track.

## Sources of Truth

- The working source of truth for the current implementation is `OPSPULSE_V2_STAGE_TRACKER.md`.
- `OPSPULSE_V1_STAGE_TRACKER.md` documents the closed V1 implementation and decisions.

## What This Repo Contains

- A FastAPI incident API.
- A FastAPI RAG/analysis service.
- A React frontend dashboard.
- Shared contracts, mock data, and stage-tracked implementation notes.
